## 📝 画像の追加方法

1. 適切なディレクトリに画像ファイルを配置
2. ファイル名は英語、小文字、ハイフン区切りで統一
3. README.mdの該当セクションで画像を参照
   ```markdown
   ![説明文](images/filename.png)
   ```
